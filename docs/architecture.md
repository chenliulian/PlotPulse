## 核心组件

### 1. Agent系统

每个Agent负责创作流程的一个环节：

- **PlotAgent** - 情节设计，输出大纲和情节点
- **CharacterAgent** - 角色设计，创建角色档案
- **WorldbuildingAgent** - 世界观构建，设定故事背景
- **WritingAgent** - 章节写作，生成正文内容
- **EditingAgent** - 编辑润色，提升文本质量
- **ReviewerAgent** - 内容审阅，评估和改进建议

### 2. 流水线系统

- **NovelPipeline** - 管理整本小说的创作流程
- **ChapterPipeline** - 管理单章的创作流程

### 3. 模型抽象

- **BaseModel** - 模型接口基类
- **ModelManager** - 模型管理器，支持多模型切换

## 数据流

```
用户输入 → 创建项目 → 生成大纲 → 设计角色 → 构建世界观 → 写作章节 → 编辑润色 → 审阅 → 成品输出
```

## 扩展性设计

1. **Agent扩展** - 继承 `NovelAgent` 基类即可创建新Agent
2. **模型扩展** - 实现 `BaseModel` 接口即可接入新模型
3. **工具扩展** - 在 `tools` 模块添加新功能
