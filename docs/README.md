## 简介

PlotPulse 是一个基于AI的小说创作框架，通过多Agent协作完成从构思到成品的完整创作流程。

## 核心特性

- 🤖 **多Agent协作** - 情节、角色、世界观、写作、编辑、审阅各司其职
- 📝 **完整创作流程** - 从大纲到成品的端到端解决方案
- 🎨 **可扩展架构** - 易于添加新的Agent和功能
- 📚 **作品管理** - 系统化的小说项目管理
